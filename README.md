EMI Calculator
Description

This is a Python-based EMI (Equated Monthly Installment) Calculator that helps users calculate the monthly EMI for a loan based on:

Principal Loan Amount
Annual Interest Rate
Number of Monthly Payments

The program also validates user inputs to ensure that only positive values are accepted.


Features--:
Accepts loan amount as input
Accepts annual interest rate
Accepts number of monthly payments
Validates user input
Calculates monthly EMI
Displays results in a user-friendly format
